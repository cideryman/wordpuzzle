# 퍼즐 메이커

브라우저에서 바로 실행할 수 있는 퍼즐 앱입니다.  
단어 찾기와 미로 찾기를 만들고, 화면에서 직접 풀거나 A4로 인쇄/PDF 저장할 수 있습니다.

## 주요 기능

### 단어 찾기

- 단어 목록 직접 입력
- 격자 크기 선택: 10x10, 12x12, 14x14, 16x16
- 방향 난이도 선택
  - 가로/세로
  - 가로/세로/대각선
  - 역방향 포함
- 랜덤 퍼즐 생성
- 화면에서 드래그로 단어 선택
- 찾은 단어 자동 표시

### 미로 찾기

- 난이도 선택
  - 쉬움
  - 보통
  - 어려움
  - 매우 어려움
- 난이도에 따라 미로 크기 변경
- `매우 어려움`은 100x100 미로 생성
- 화면 버튼 또는 키보드 방향키로 플레이
- 이동한 길 표시
- A4 인쇄 및 PDF 저장

### 공통

- A4 인쇄 및 PDF 저장
- 모바일에서는 플레이 모드 중심 표시

## 실행 방법

설치 과정이 필요 없습니다.

1. `index.html` 파일을 브라우저에서 엽니다.
2. `단어 찾기` 또는 `미로 찾기` 모드를 선택합니다.
3. 설정을 고릅니다.
4. `새 퍼즐 생성`을 누릅니다.
5. 화면에서 플레이하거나 `인쇄 / PDF 저장`으로 출력합니다.

## 미로 조작

- 방향키: 이동
- 화면의 화살표 버튼: 이동
- `플레이 초기화`: 시작 위치로 되돌리기

## 모바일 사용

아이패드나 스마트폰에서도 사용할 수 있습니다.  
모바일에서는 인쇄 미리보기와 인쇄 버튼이 숨겨지고, 플레이 모드만 크게 표시됩니다.

## PWA 설치

GitHub Pages처럼 HTTPS로 배포하면 PWA로 설치할 수 있습니다.

- Android Chrome: 메뉴에서 `홈 화면에 추가` 또는 `앱 설치`
- iPhone/iPad Safari: 공유 버튼에서 `홈 화면에 추가`
- PC Chrome/Edge: 주소창의 설치 아이콘 또는 메뉴의 `앱 설치`

서비스 워커는 `file://`로 직접 열 때는 등록되지 않고, HTTPS 또는 localhost 환경에서 동작합니다.

## 인쇄 및 PDF 저장

PC에서 `인쇄 / PDF 저장` 버튼을 누르면 브라우저 인쇄 창이 열립니다.

- 실제 프린터 선택: A4 용지 인쇄
- PDF로 저장 선택: PDF 파일 저장
